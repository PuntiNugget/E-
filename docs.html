E# Language Reference

Welcome to the complete documentation for E# (English Sharp) Game Engine Edition! E# reads just like a book.

1. Variables & Math

  • Create a Variable: set [number|string|boolean] [name] to [value]
    set number score to 0
    set string playerName to "Hero"
    set boolean isRunning to true
  • Random Numbers: set number [name] to random between [min] and [max]
  • Combine Strings: combine [value1] and [value2] into [newName]
    combine "Score: " and score into scoreText
  • Math: add [number] to [variable] or subtract [number] from [variable]

2. Game Objects (The Good Stuff!)

  • Create Object: set object [name] to [sprite] at x [num] y [num]
    set object player to "🚀" at x 150 y 150
  • Draw Object: draw object [name]
  • Move Object Coordinates: add [num] to [object]'s x (or y)
  • Enemy Tracking (AI): move [object1] towards [object2] by [speed]
    move enemy towards player by 2
  • Collision Detection: if [object1] is touching [object2]
    if player is touching enemy
        print "You died!"
    end if

3. Graphics & UI

  • Draw Sprite (No Object): draw sprite [text/emoji] at x [num] y [num]
  • Draw Text: draw text [string/variable] at x [num] y [num]
    draw text "Level 1" at x 10 y 30
  • Clear Screen: clear canvas

4. Input & Control

  • Check Keys (Real-time): if you pressed "[key]"
    if you pressed "arrow up"
        subtract 5 from player's y
    end if
  • Terminal Input (Pause): ask user for [string|number] [name] with "[prompt]"
Special Key Mappings:
Use these names inside your quotes for keyboard inputs:
"arrow up", "arrow down", "arrow left", "arrow right", "space", "w", "a", "s", "d".

5. Logic & Loops

  • If Statements: if [value] is [equal to|greater than|less than] [value]
  • Boolean Checks: if [variable] is [true|false]
  • While Loops: while [variable] is [true|false]
    Don't forget to put wait 30 milliseconds inside your game loops so the browser doesn't crash!
  • Always close logic blocks with end if or end while!

6. Functions

  • Define a Function: Wraps code so it can be reused later.
    create function movePlayer
        add 5 to player's x
    end function
  • Execute a Function: run [functionName]

7. Utilities

  • Timers: wait [num] seconds or wait [num] milliseconds
  • Printing: print [text or variable]
  • Clearing Terminal: clear terminal
  • Quit: end program
