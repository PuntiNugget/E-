E# Language Reference

Welcome to the official documentation for E# (English Sharp). E# is a strongly-typed, asynchronous programming language designed to read like plain English.

Variables & Data Types

E# requires you to declare the type of a variable before naming it. The supported types are number, string, and boolean.

set number age to 25
set string greeting to "Hello World"
set boolean isReady to true

Math & Random Numbers

You can perform basic arithmetic on number variables, or generate random numbers.

add 5 to age
subtract 2 from age

// Generates a random number inclusive of the min and max
set number diceRoll to random between 1 and 6

Input & Output

Interact with the terminal using print commands and user prompts.

print "System Online"
print age

ask user for string playerName with "What is your name?"
clear terminal

Logic & Loops

Control the flow of your program based on conditions. Supported comparisons: equal to, greater than, less than.

if age is greater than 18
    print "Access Granted"
end if

while age is less than 30
    add 1 to age
end while

Program Flow & Windows

Pause execution, wait for user key presses, or open new web pages.

wait 2 seconds
wait for key "Enter"
open window "https://example.com"
end program
