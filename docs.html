E# Engine: Documentation (v1.6)

Welcome to the official E# documentation. Use this guide to look up commands, variable types, and game logic systems.

1. Variables & Data Types

Command SyntaxDescriptionExample
set number [name] to [value]Creates/updates a number.set number score to 0
set string [name] to "[text]"Creates/updates text.set string msg to "Hello"
set boolean [name] to [true/false]Creates/updates a true/false flag.set boolean isJumping to false
set vector2 [name] to x [n] y [n]Stores a 2D coordinate.set vector2 spawnPoint to x 10 y 50
set [name] to [value]Updates an existing variable.set score to 100

2. Math & Logic

Command SyntaxDescriptionExample
add [val] to [var]Addition.add 1 to score
subtract [val] from [var]Subtraction.subtract 5 from player's x
multiply [var] by [val]Multiplication.multiply player's scale by 2
divide [var] by [val]Division.divide speed by 2
combine [str1] and [str2] into [var]Joins strings together.combine "Score: " and score into uiText

3. Game Objects & Properties

Command SyntaxDescriptionExample
set object [name] to [sprite] at x [n] y [n]Creates a game object.set object player to "🚀" at x 100 y 100
set [name]'s [property] to [value]Changes x, y, scale, or rotation.set player's scale to 2.5
draw object [name]Renders the object.draw object player
move [obj1] towards [obj2] by [speed]Moves toward a target.move zombie towards player by 3

4. Spawners & Groups

Command SyntaxDescriptionExample

for each [item] in [group]Loops through every item.for each laser in lasers
remove [item] from [group]Destroys an object.remove laser from lasers

💡 Pro-Tips for E# Developers
  • Frame Rates: Use wait 16 milliseconds at the end of your main while loop for ~60 FPS.
  • Memory Management: Always use remove [item] from [group] when spawned objects leave the screen to prevent crashing.
  • Layering: The engine draws in the order you write. To put the player on top, draw object player must come after draw tilemap.
