E# Language Documentation

Welcome to the official documentation for E# v1.5! Below is a comprehensive list of all commands available in the engine.

1. Variables & Math Updated!

  • set number score to 0 (Creates a number)
  • set string name to "Hero" (Creates text)
  • set boolean isAlive to true (Creates true/false)
  • set score to 10 (Updates an existing variable)
  • set number rand to random between 1 and 10
  • add 5 to score
  • subtract 2 from score
  • multiply score by 2 New!
  • divide score by 2 New!

2. Vector2 Math New!

  • set vector2 velocity to x 5 y 10 (Creates a 2D vector)
  • add velocity to player (Adds vector to object position)
  • subtract velocity from player
  • multiply velocity's x by -1 (Modify vector directly)

3. Objects & Transforms Updated!

  • set object player to "🚀" at x 100 y 150 (Spawns object)
  • set player's x to 200
  • set player's y to 50
  • set player's scale to 2 New!
  • set player's rotation to 90 New!
  • add 5 to player's rotation
  • move player towards enemy by 5 (Auto-chase mechanic)

4. Pixel Art & Tilemaps New!

You can use the built-in IDE editors to generate code for Sprites and Tilemaps. Use `W` (White), `R` (Red), `B` (Blue), `G` (Green), `Y` (Yellow), `K` (Black), and `.` (Transparent).

create pixelart playerSprite ".RRRRRR." "RWWWWWWW" ".RR..RR." end pixelart set object player to playerSprite at x 100 y 100

5. Input & Rendering

  • if you pressed "arrow right"
  • if you pressed "space"
  • clear canvas
  • draw object player
  • set color to "red" (Changes text/drawing color)
  • draw text "Game Over" at x 150 y 150

6. Collisions & Physics

  • if player is touching enemy (AABB Box collision)
  • if player is touching tile "🟥" in myLevel (Tilemap collision)
  • if raycast from player to enemy hits "⬛" in myLevel (Line-of-sight check)
