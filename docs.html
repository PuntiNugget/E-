E# Language Reference (Combat Engine Edition)

Welcome to the complete documentation for E# (English Sharp) Game Engine Edition! Build 2D action games simply by typing.

1. Variables & Object Manipulation

  • Create a Variable: set [number|string|boolean] [name] to [value]
  • Create Object: set object [name] to [sprite] at x [num] y [num]
  • Set Exact Coordinates: set [object]'s [x|y] to [value]
    *New Feature: You can set a coordinate to match another object!
    set laser's x to player's x

2. Movement & Graphics

  • Move Object Coordinates: add [num] to [object]'s x (or y)
  • Enemy Tracking (AI): move [object1] towards [object2] by [speed]
  • Draw Object: draw object [name]
  • Clear Screen: clear canvas

3. Advanced Collisions & Raycasting

  • Object vs Object: if [object1] is touching [object2]
  • Object vs Wall: if [object] is touching tile "[emoji]" in [tilemap]
  • Line of Sight (Raycasting): Checks if there is a clear path between two objects, looking for a specific tile blocking the way. Use hits or does not hit.
    if raycast from enemy to player does not hit "🟥" in myLevel move enemy towards player by 2 end if

4. Tilemaps

  • Create a Tilemap: Define a grid of emojis to act as your game background. (Or use the Map Editor Tab!)
  • Draw Tilemap: draw tilemap [name] at x [num] y [num]

5. Logic & Control

  • While Loops: while [variable] is [true|false]
    Note: Always put wait 30 milliseconds inside your game loops!
  • Functions: Use create function [name] and run [name].
  • Check Keys: if you pressed "[key]" (Supports: "arrow up", "space", etc.)
