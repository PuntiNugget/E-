E# Language Reference (Engine Edition)

Welcome to the complete documentation for E# (English Sharp) Game Engine Edition! Build 2D worlds simply by typing.

1. Variables & Objects

  • Create a Variable: set [number|string|boolean] [name] to [value]
  • Create Object: set object [name] to [sprite] at x [num] y [num]
    set object player to "👽" at x 150 y 150

2. Movement & Graphics

  • Move Object Coordinates: add [num] to [object]'s x (or y)
  • Enemy Tracking (AI): move [object1] towards [object2] by [speed]
  • Draw Object: draw object [name]
  • Draw Text: draw text [string] at x [num] y [num]
  • Clear Screen: clear canvas

3. Tilemaps & Visual Editor

  • Create a Tilemap: Define a grid of emojis to act as your game background or level. (Or use the built-in Map Editor Tab to do this for you!)
    create tilemap myLevel "🟥🟥🟥" "🟥🟩🟥" "🟥🟥🟥" end tilemap
  • Draw Tilemap: draw tilemap [name] at x [num] y [num]
    Note: Always draw your tilemap before your objects so your character stands on top of the map!

4. Game Physics & Map Collisions

  • Object vs Object: if [object1] is touching [object2]
  • Object vs Wall (Tilemap Physics): To make a tile "solid", check if the player hit it immediately after moving. If they did, undo the movement!
    if you pressed "arrow right" add 5 to player's x if player is touching tile "🟥" in myLevel subtract 5 from player's x end if end if

5. Audio & Control

  • Play a Sound: play sound "[url]"
  • Check Keys: if you pressed "[key]" (Supports: "arrow up", "arrow down", "arrow left", "arrow right", "space", letters, and numbers).

6. Logic & Loops

  • While Loops: while [variable] is [true|false]
    Note: Always put wait 30 milliseconds inside your game loops so the browser doesn't freeze!
  • Functions: Wraps code so it can be reused. Use create function [name], and trigger it with run [name].
